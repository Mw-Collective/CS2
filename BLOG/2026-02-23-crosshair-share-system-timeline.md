# Crosshair Share System Timeline

## Source-Backed Facts

- Counter-Strike release notes on 2020-02-24 introduced crosshair sharing and import support.
- Counter-Strike release notes on 2020-04-16 added support to view and copy a crosshair from observed players.

## Inference for This Repo

- Inference: Crosshair documentation should track both share codes and context (map type, role, and visibility notes), not only raw codes.
- Inference: Observed-player copying makes periodic crosshair review useful during demo or match analysis.
