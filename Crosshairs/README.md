# Crosshair Notes

This folder tracks share-code presets and review notes.

## Source-Backed Background

- Valve release notes (2020-02-24) added support for sharing and importing crosshair codes.
- Valve release notes (2020-04-16) added support for viewing and copying crosshairs from observed players.

## Repository Convention

- Keep each preset in markdown format.
- Include source, style notes, and last validation date.
- Start from: [codes-template.md](codes-template.md)
- Track observed-player copies in: [observed-crosshair-log-template.md](observed-crosshair-log-template.md)
