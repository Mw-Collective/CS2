# Observed Crosshair Log Template

Use this template when copying crosshairs from watched players.

## Entry

- Date: `YYYY-MM-DD`
- Player/source: `NAME OR LINK`
- Match/demo: `EVENT OR LINK`
- Imported code: `PASTE_CODE`
- Visibility notes: `SHORT NOTES`
- Aim style context: `RIFLE / AWP / MIXED`
